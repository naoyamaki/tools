`gwt new [branch_name]`コマンドの実行結果
  今いるディレクトリに.gitディレクトリがある（gitリポジトリ直下）であることを確認
  現在のディレクトリ名をrepo_nameという変数に格納
  未commitの変更差分がある場合stash_flugを立ててgit stash -uを実行
  branch_nameに/がある場合/を-に置き換えた文字列をbranch_name_dirという変数に格納
  ../[branch_name_dir]/[repo_name]という形式ででワークツリー作成（`git worktree add -b branch_name ../branch_name_dir/repo_name`）
  stash_flugが立っている場合、git stash popを実行

`gwt new [branch_name] --recursive [dir_1] [dir_2]...`コマンドの実行結果
  dir_1に移動
  `gwt new [branch_name]`コマンドでやっていることを実行
  dir_2に移動
  `gwt new [branch_name]`コマンドでやっていることを実行
  dir_3に移動、、、繰り返し

`gwt rm [branch_name]`コマンドの実行結果
  git worktree list | awk '$3=="[[branch_name]]" {print $1}'の結果をtrget_dirとする
  git worktree remove -f trget_dir
`gwt rm -d [branch_name]`コマンドの実行結果
  `gwt rm [branch_name]`コマンドの処理を実行
  git br -D [branch_name]実行
